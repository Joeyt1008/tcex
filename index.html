Overview¶

The TcEx Class provides method for common functionality used in ThreatConnect Exchange or ThreatConnect Playbook apps.

Features¶

  • Automatic authorization for HMAC and Token base authentication when using Resources functionality. This allow the app to work seemlessly on both ThreatConnect Exchange and the ThreatConnect Integration Server.
  • App logger that supports both filesystem and API logging (Logging).
  • Batch add through job interface (Jobs).

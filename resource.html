Resources¶

The TcEx() Class provides access to the ThreatConnect API through instantiation of a Resource Class in the tcex.resource property.

Indicator Resources¶

Resource Type Class
Indicator (Base) Indicator
Address Address
EmailAddress EmailAddress

Note

Custom Indicators can be accessed by the Type. The ThreatConnect platform support Custom indicator types with a space in the name. To prevent issues with the space all Custom Resource (Indicator) types should be made safe by using the safe_rt() method.

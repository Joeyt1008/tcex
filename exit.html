Exit¶

All ThreatConnect Exchange and Playbook apps should deliberately provide an exit code on execution completion or failure. The TcEx module provides the exit() and exit_code() methods to handle exit codes.

Supported Exit Codes¶

  • 0 - Successful execution of App
  • 1 - Failed execution of App
  • 3 - Parital failure of App

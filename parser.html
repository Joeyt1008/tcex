Parser / Args¶

The TcEx() module provides the parser() property which returns and instance of argparser(). This argparser instance is an extension of the Python argparser method with predefined arguments specifically for ThreatConnect Exchange apps..

Default Arguments¶

Section Values
API _api_arguments()
Batch _batch_arguments()
Playbook _playbook_arguments()
Standard _standard_arguments()

Custom Argument Example¶

from tcex import TcEx

tcex = TcEx()
parser = tcex.parser
parser.add_argument('--myarg', help='My Custom Argument', required=True)
args = tcex.args
